# Privacy Policy

This is a minimal privacy statement for the development microservice.

- Data collected: user fitness metrics (steps, calories, sleep, heart-rate, weight).
- Storage: data is stored in the configured Firebase project.
- Retention: for development, data is retained until manually removed.
- Sharing: data is not shared with third parties by this project.

If you deploy this for others, implement clear consent flows, access controls,
retention schedules, and a way for users to delete their data.
